# Predict Her

This is a professional, AI-powered health tracking and PCOD risk assessment application built with React and Vite.

## Features
- **Dashboard**: Real-time health metrics and cycle tracking.
- **AI Insights**: PCOD risk analysis and personalized health protocols.
- **Community**: Anonymous forum for health discussions.
- **Logging**: Detailed symptom and lifestyle logging.

## Tech Stack
- Frontend: React, Vite
- Styling: Vanilla CSS (Theme: Deep Purple & Midnight Blue)
- Backend: Firebase (Authentication & Database)
- Icons: Lucide React

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
